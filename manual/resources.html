Relevant Online Resources

  • SVG Essentials (an insightful book taking you through the ins and outs of SVG)
  • Unicode Lookup (to lookup Unicode and HTML special characters by name and number)
  • LaTeX Previewer (a user-friendly web utility for generating LaTeX output in various formats)
  • W3C Validator (to check the markup validity of Web documents in HTML, SVG, MathML, etc.)
  • Image Converter (conversion between a variety of target formats, such as SVG, EPS, and PDF)
  • JaxoDraw (a Java program with graphical user interface for drawing Feynman diagrams)
  • PyFeyn (a Python-language based system for drawing Feynman diagrams)
  • MathJax (an open-source JavaScript display engine for mathematics in all browsers)
  • Ry’s MathML Tutorial (a comprehensive coverage of MathML’s Presentation Markup)
  • HTML5 for Publishers (an intro to modern Web technologies to enhance your ebook content)
  • Mozilla Developer Network (docs and tutorials on HTML, CSS, JavaScript, SVG, MathML, etc.)
